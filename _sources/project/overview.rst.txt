OpenKinetics Predictor
======================

OpenKinetics Predictor is a web service for enzyme kinetic parameter prediction.

You submit protein sequences and substrate data. The service returns structured predictions from published machine learning models.

Main use cases
--------------

- Estimate kcat from protein sequence and substrate SMILES.
- Estimate Km from protein sequence and substrate SMILES.
- Estimate kcat/Km with a supported method.
- Compare predictions across published methods.
- Run batch prediction jobs through the web interface or REST API.

Main users
----------

- Enzyme engineers screening candidate enzymes.
- Modellers building kinetic models.
- Researchers comparing kinetic prediction methods.
- Developers adding prediction engines.
